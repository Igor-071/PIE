# PRD Sections Reference

Complete list of all PRD sections generated by the Product Intelligence Engine.

---

## Tier 1: Technical Extraction (Automated)

These sections are extracted directly from the codebase:

1. **Project** - Basic project metadata
2. **Screens** - UI screens/pages detected
3. **Components** - React/UI components
4. **Navigation** - Navigation structure and routing
5. **Data Model** - Database schemas and data structures
6. **API Endpoints** - REST/GraphQL endpoints
7. **State Management** - Global state patterns
8. **Events** - User events and analytics
9. **AI Metadata** - Extraction quality and assumptions

---

## Tier 2: Strategic Analysis (AI-Generated)

Strategic and business-focused sections:

10. **Brand Foundations** - Mission, vision, core values
11. **Target Audience** - User personas and demographics
12. **Problem Definition** - Problem statement and business impact
13. **Solution Overview** - Value proposition and key features
14. **Lean Canvas** - Business model canvas
15. **Customer Profiles** - Early adopters and ideal customers
16. **Positioning & Messaging** - Market positioning

---

## Tier 3: Detailed Requirements (AI-Generated)

### Core Sections (Original 11)

17. **Goals & Success Criteria** - Business objectives and KPIs
18. **MVP Scope** - Features in/out of scope
19. **Assumptions** - Technical, operational, financial assumptions
20. **Dependencies** - External services and operational dependencies
21. **Role Definition** - User roles and access control
22. **Product Requirements** - Detailed acceptance criteria by module
23. **Critical User Flows** - Step-by-step user journeys
24. **Technical Requirements** - Infrastructure and technical specs
25. **Non-Functional Requirements** - Performance, security, scalability
26. **Risk Management** - Risks and mitigation strategies
27. **Open Questions** - Unresolved questions for stakeholders

### Phase 1 Additions (NEW - 3 sections)

28. **üÜï Executive Summary** - C-level summary of entire PRD
    - Overview and problem statement
    - Solution highlights
    - Key differentiators
    - Target market
    - Strategic goals
    - Technical approach (business terms)
    - Risk overview
    - Vision statement

29. **üÜï Competitive Intelligence** - Market positioning analysis
    - Market category identification
    - Likely competitors (inferred)
    - Competitive advantages
    - Market positioning strategy
    - Pricing strategy hints
    - Go-to-market considerations
    - White space opportunities

30. **üÜï Implementation Roadmap** - Phased delivery plan
    - 3-phase breakdown (MVP, Core, Enhancement)
    - Weekly milestones per phase
    - Team size requirements
    - Deliverables per phase
    - Cost estimates (labor + infrastructure)
    - Timeline dependencies
    - Risk factors

## Custom Section Control

Skip sections you don't need:

```typescript
const result = await runTier3Agent(prdJson, evidence, tier1Data, {
  skipSections: [
    'executiveSummary',           // Skip if internal doc only
    'competitiveIntelligence',    // Skip if stealth mode
    'implementationRoadmap'       // Skip if timeline not needed
  ]
});
```
